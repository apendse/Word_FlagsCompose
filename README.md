This is a game that where user guesses the country based on the image of flag shown. 
The game does not need any network. It can be played without any network.
The game tries to make questions more interesting. It keeps track of which flags did you guess correctly and which you guessed incorrectly. The questions which a user misses are asked 
more frequently than the ones which are answered correctly. 
There are confusing answers which are picked with more likelihood. e.g. Liberia, Malaysia and the USA flags are similar. India and Niger flags are similar. Poland and Indonesia flags 
are similar. So when user is asked to guess the flag of Poland, Indonesia is very likely to be a wrong answer presented.
